# Source Code Review

Burst is built with WXT, React, TypeScript, and Bun.

## Rebuild Firefox package

```sh
bun install --frozen-lockfile
bun run zip:firefox
```

The Firefox submission is generated by WXT with:

```sh
wxt zip -b firefox
```

This creates the extension ZIP and a source ZIP under `.output/`.

## Rebuild Chrome package

```sh
bun install --frozen-lockfile
bun run zip
```

The Chrome Web Store ZIP is generated by WXT with:

```sh
wxt zip
```

## Verification commands

```sh
bun run compile
bun test
bun run build:firefox
bun run build:extension
```
